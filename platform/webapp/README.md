# GMZ WebApp (Control Plane UI)

## Stack (festgelegt)
- Next.js + TypeScript
- TailwindCSS + shadcn/ui
- Mobile-first responsive Layout

## Kernmodule
1. Setup Wizard (Management VM Erstinstallation)
2. Tenant Wizard (Sizing, VLAN, Auth, Apps)
3. Deployments (Jobs, Logs, Status)
4. Monitoring (Grafana Embeds + Health Cards)
5. Reporting (PDF/CSV Export)
6. RBAC + Audit Log
7. Secrets/Connector Management (Entra/LDAP)

## UX Leitlinien
- Kundenzentrierte Navigation
- Wenige Schritte, klare Fortschrittsanzeige
- Desktop/Tablet/Mobile optimiert
- Designsystem mit GMZ Branding (Farben/Logo/Favicon)
